# 구글 애드센스 설정 가이드

## 1. 애드센스 승인을 위한 필수 작업

### ✅ 완료된 작업
- [x] 필수 페이지 생성 (개인정보처리방침, 이용약관, 문의하기)
- [x] 텍스트 콘텐츠 추가 (가이드 섹션 8개 글)
- [x] ads.txt 파일 생성
- [x] AdsUnit 컴포넌트 생성
- [x] layout.tsx에 애드센스 스크립트 추가
- [x] 병원 상세 페이지에 광고 배치

## 2. 환경 변수 설정

`.env.local` 파일에 다음을 추가하세요:

```bash
NEXT_PUBLIC_ADSENSE_CLIENT_ID=ca-pub-YOUR_PUBLISHER_ID
```

**주의**: `YOUR_PUBLISHER_ID`를 실제 애드센스 발행자 ID로 교체하세요.

## 3. ads.txt 파일 설정

`public/ads.txt` 파일을 열고 구글 애드센스에서 발급받은 코드를 입력하세요:

```
google.com, pub-YOUR_PUBLISHER_ID, DIRECT, f08c47fec0942fa0
```

## 4. 광고 슬롯 ID 설정

병원 상세 페이지(`app/hospitals/[id]/page.tsx`)에서 다음을 교체하세요:

- `YOUR_SLOT_ID_1`: 상단 광고 슬롯 ID
- `YOUR_SLOT_ID_2`: 비용 정보 위 광고 슬롯 ID  
- `YOUR_SLOT_ID_3`: 비용 정보 아래 광고 슬롯 ID

애드센스 대시보드에서 광고 단위를 생성한 후 슬롯 ID를 복사하여 사용하세요.

## 5. 광고 배치 위치

### 병원 상세 페이지
1. **상단**: 병원 이름 바로 아래 (클릭률 최고)
2. **비용 정보 위**: TreatmentPriceDashboard 바로 위
3. **비용 정보 아래**: TreatmentPriceDashboard 바로 아래

### 메인 페이지
- 가이드 섹션에 자동 광고가 표시됩니다 (Auto Ads)

## 6. 애드센스 승인 프로세스

1. **사이트 제출**: 구글 애드센스에 사이트 URL 제출
2. **심사 대기**: 보통 1-2주 소요
3. **승인 후**: 광고 코드 활성화

## 7. 승인 후 확인 사항

- [ ] ads.txt 파일이 올바르게 접근 가능한지 확인 (`https://yourdomain.com/ads.txt`)
- [ ] 광고가 정상적으로 표시되는지 확인
- [ ] 모바일/데스크톱 모두에서 광고 확인
- [ ] Auto Ads 설정 확인

## 8. 수익 최적화 팁

1. **광고 필터링**: 애드센스 콘솔에서 게임, 성인물 등 부적절한 광고 차단
2. **위치 분석**: 어떤 위치의 광고가 클릭률이 높은지 분석
3. **콘텐츠 강화**: 가이드 글을 더 추가하여 텍스트 비율 높이기
4. **사용자 경험**: 광고가 콘텐츠를 방해하지 않도록 배치

## 9. 트러블슈팅

### 광고가 표시되지 않을 때
- 환경 변수 확인 (`NEXT_PUBLIC_ADSENSE_CLIENT_ID`)
- 브라우저 콘솔에서 에러 확인
- 애드센스 승인 상태 확인

### ads.txt 접근 불가
- `public/ads.txt` 파일 위치 확인
- 빌드 후 배포 확인
- Vercel에서 정적 파일 서빙 확인
